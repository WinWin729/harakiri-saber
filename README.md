# ハラキリセイバー(試作之伍)

切腹しに行く騎士が、命を捨てる覚悟を武器に、命を惜しむ銃士たちを斬り抜けるデッキ構築ローグライク。

**HPは通貨。血を流すほど強くなる。**

## 遊ぶ

GitHub Pages 公開後、ここに URL を記載:
`https://<ユーザー名>.github.io/harakiri-saber/`

ブラウザだけで動きます(スマホ可・音が出ます)。単一HTMLのプロトタイプです。

## テスターの方へ

知りたいことは4つだけです。

1. **面白かったか**(また遊びたいと思ったか)
2. **分からなかったこと**(ルール・表示・用語、何でも)
3. **理不尽だと感じた死**があったか
4. プレイ時間と、勝てたか

感想は X の `#ハラキリセイバー` か、配布者まで直接どうぞ。

## 最低限のルール

- カードを使って戦う。気力が足りなければ **HP(血)で払える**
- 攻撃カードは**踏み込む**。踏み込むと銃火を浴びる ── その血が**覚悟**になる
- 覚悟が「位三」に達すると必殺《介錯一閃》が手札に宿る
- 防御カードは間を切る。遠いほど守りが固く、近いほど刃が重い
- 敵の攻撃を**ぴったり同値のブロック**で受けると「見切り」
